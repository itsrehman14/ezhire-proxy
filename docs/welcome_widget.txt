=====================================================================
Welcome Actions Widget
=====================================================================

Used by: WelcomeAgent
Context: Shown after the car carousel on session start, providing quick
action buttons that map to available agents.

WIDGET TEMPLATE

```tsx
<Card size="sm" padding={0} background="#1a1a1a" theme="dark">
  <Col padding={8} gap={6}>
    <Col gap={1}>
      <Text value="What else can I help with?" size="md" weight="semibold" color="primary" />
      <Caption value="Tap an option to get started" color="secondary" />
    </Col>

    <ListView direction={"row"} scrollable maxWidth="100">
      <Row gap={4}>
        {items.map((item) => (
          <ListViewItem
            key={item.title}
            radius="lg"
            height={90}
            width={200}
            wrap="wrap"
            minWidth={200}
            minHeight={90}
            background="#222D34"
            onClickAction={{ type: "reply", payload: { text: item.reply_text } }}
          >
            <Row>
              <Col gap={0} flex="auto">
                <Text value={item.title} size="md" weight="semibold" maxLines={2} color="purple-200" />
                <Caption value={item.subtitle} size="sm" color="primary" maxLines={2} />
              </Col>
            </Row>
          </ListViewItem>
        ))}
      </Row>
    </ListView>
  </Col>
</Card>
```

WIDGET SCHEMA

```tsx
import { z } from "zod"

const QuickAction = z.strictObject({
  title: z.string(),
  subtitle: z.string(),
  reply_text: z.string(),
})

const WidgetState = z.strictObject({
  items: z.array(QuickAction),
})

export default WidgetState
```
