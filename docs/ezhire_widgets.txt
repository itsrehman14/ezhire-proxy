eZhire Widgets
==============

Widgets for the eZhire car rental agents. Each widget includes a template (JSX),
schema (Zod), and example data (JSON).

All widgets use a dark theme (background="#1a1a1a", theme="dark") with alpha-5
accents for icon containers and inner sections.


=====================================================================

WIDGET 2: Rental Cars Carousel
=====================================================================

Used by: CarBrowsingAgent, BookingAgent, GetRentalCarsAgent, GeneratePreBookingAgentV2
Context: Shown when presenting multiple available cars in a horizontal carousel.

WIDGET TEMPLATE

```tsx
<Card size="full" padding={10} background="#1a1a1a" theme="dark">
  {/* Header Section */}
  <Col padding={{ left: 4, top: 6, bottom: 2 }} gap={1}>
    <Text value="Available Vehicles" size="lg" weight="semibold" color="primary" />
    <Text value="Browse our premium fleet" size="xs" color="secondary" />
  </Col>

  <ListView scrollable>
    <Row gap={4} padding={{ left: 4, right: 4, bottom: 6 }} wrap="nowrap">
      {cars.map((item) => (
        <ListViewItem
          key={item.id}
          onClickAction={{
            type: "reply",
            payload: { text: `I'd like to book the ${item.name}` }
          }}
        >
          <Box
            width={300}
            background="#212121"
            radius="xl"
            border={{ size: 1, color: "subtle" }}
          >
            {/* Image Container */}
            <Box height={160} background="alpha-5" padding={10} radius={{ top: "xl" }}>
              <Image src={item.image} fit="contain" flush />
            </Box>

            {/* Content Container */}
            <Col padding={10} gap={4}>
              {/* Header: Name (Left) and Category Badge (Right) */}
              <Row align="center">
                <Text value={item.name} size="md" weight="bold" maxLines={1} truncate />
                <Spacer />
                <Badge label={item.category} variant="soft" color="secondary" size="sm" />
              </Row>

              {/* Pricing Grid */}
              <Row justify="between">
                <Col gap={0.5}>
                  <Text value={item.dailyRate.split(' ')[0]} size="sm" weight="bold" />
                  <Caption value="Daily" color="secondary" />
                </Col>
                <Col gap={0.5}>
                  <Text value={item.weeklyRate.split(' ')[0]} size="sm" weight="bold" />
                  <Caption value="Weekly" color="secondary" />
                </Col>
                <Col gap={0.5}>
                  <Text value={item.monthlyRate.split(' ')[0]} size="sm" weight="bold" />
                  <Caption value="Monthly" color="secondary" />
                </Col>
              </Row>

              {/* Specs Footer - Aligned Left with Pricing */}
              <Row gap={4} align="center">
                <Row gap={1} align="center">
                  <Icon name="user" size="sm" color="secondary" />
                  <Text value={item.seats} size="xs" color="secondary" />
                </Row>
                <Row gap={1} align="center">
                  <Icon name="suitcase" size="sm" color="secondary" />
                  <Text value={item.luggage} size="xs" color="secondary" />
                </Row>
              </Row>

              {/* Centered Select Button */}
              <Row justify="center">
                <Button
                  label="Select Vehicle"
                  block
                  variant="outline"
                  size="md"
                  radius="full"
                  onClickAction={{
                    type: "reply",
                    payload: { text: `I'd like to book the ${item.name}` }
                  }}
                />
              </Row>
            </Col>
          </Box>
        </ListViewItem>
      ))}
    </Row>
  </ListView>
</Card>
```

WIDGET SCHEMA

```tsx
import { z } from "zod"

const Car = z.strictObject({
  id: z.string(),
  image: z.string(),
  name: z.string(),
  category: z.string(),
  dailyRate: z.string(),
  weeklyRate: z.string(),
  monthlyRate: z.string(),
  seats: z.string(),
  luggage: z.string(),
})

const WidgetState = z.strictObject({
  cars: z.array(Car),
})

export default WidgetState
```

WIDGET 3: Pickup Location Card
=====================================================================

Used by: CarBrowsingAgent, BookingAgent, GetPickupLocationsAgent, GeneratePreBookingAgentV2
Context: Shown when a specific pickup location is selected or highlighted.

WIDGET TEMPLATE

```tsx
<Card size="sm" padding={0} background="#1a1a1a" theme="dark">
  {/* Card padding pushed to 10 for a high-end, airy feel */}
  <Col padding={10} gap={10}>
    {/* Header Section with extra breathing room */}
    <Col gap={2}>
      <Text value="Locations" size="xl" weight="bold" color="primary" />
      <Caption value="Select a branch to collect or return your vehicle" color="secondary" />
    </Col>

    <Divider flush color="subtle" />

    {/* Location List - Vertical gap pushed to 10 for maximum separation */}
    <Col gap={10}>
      {locations.map((item) => (
        <Row key={item.id} gap={6} align="center">
          {/* Larger Icon Container with its own internal air */}
          <Box 
            size={56} 
            background="alpha-10" 
            radius="lg" 
            align="center" 
            justify="center"
            border={{ size: 1, color: "subtle" }}
          >
            <Icon name="map-pin" size="lg" color="secondary" />
          </Box>

          {/* Details Section - Gap increased to 3 to separate Name from Meta */}
          <Col gap={3} flex={1}>
            <Text value={item.name} size="md" weight="bold" />
            
            {/* Metadata Row with wider dot separation */}
            <Row gap={3} align="center">
              <Caption value={item.city} color="secondary" weight="medium" />
              <Caption value="•" color="tertiary" />
              <Caption value={item.hours} color="secondary" />
            </Row>
          </Col>

          {/* Selection Button - Elevated appearance */}
          <Button
            variant="soft"
            color="primary"
            size="lg"
            iconStart="chevron-right"
            uniform
            radius="full"
            onClickAction={{
              type: "reply",
              payload: { text: `I choose ${item.name}` }
            }}
          />
        </Row>
      ))}
    </Col>
  </Col>
</Card>
```
